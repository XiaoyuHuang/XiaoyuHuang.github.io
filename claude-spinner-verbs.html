Claude Code Spinner Verbs

When Claude Code is processing your request, instead of showing a boring loading spinner, it randomly picks a verb and displays it — like Razzmatazzing… or Flibbertigibbeting… or simply Thinking… In April 2026, Claude Code's source code was leaked, and among everything inside, the most delightful discovery was this list of 187 handpicked verbs. They range from the mundane (Working, Doing) to the absurd (Discombobulating, Boondoggling) to the self-aware (Clauding). Below is the complete list, sorted alphabetically, with pronunciation and explanations in English and Chinese.
